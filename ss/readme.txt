Non-Technical Documentation/ User-guide for the End-Users of the Project

Step-1: download_vscode

Step-2: download_python

Step-3: download_arduino uno

Step-4: install libraries from Al.text files

Step-5: run the code in arduino software as following steps:

Step-6: run the code in arduino software(run the c++ code of pyfirmata)
